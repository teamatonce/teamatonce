## [Unreleased]

### Added
- Open-source release under GNU AGPL 3.0
- PostgreSQL direct database layer
- Community infrastructure (CONTRIBUTING, CODE_OF_CONDUCT, SECURITY)

- Replaced proprietary SDK with standard open-source alternatives

### Added
- AI-driven development outsourcing marketplace
- NestJS backend with 34 modules
- React frontend with Vite and Tailwind CSS
- Course management and learning paths
- AI assessments and certifications
- Escrow payment system (Stripe)
- Real-time notifications and discussions
- Search with Qdrant vector database
- Blog and content management
- Multi-language support
